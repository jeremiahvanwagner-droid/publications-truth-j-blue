# Sitemap Submission Checklist

Canonical sitemap endpoint:
- `https://pub.jeremiahvanwagner.com/sitemap.xml`

## Google Search Console
1. Open Google Search Console property for `pub.jeremiahvanwagner.com`.
2. Go to `Indexing > Sitemaps`.
3. Submit `https://pub.jeremiahvanwagner.com/sitemap.xml`.
4. Confirm status is `Success`.
5. Request indexing for:
- `https://pub.jeremiahvanwagner.com/`
- `https://pub.jeremiahvanwagner.com/publications.html`
- `https://pub.jeremiahvanwagner.com/books.html`
- `https://pub.jeremiahvanwagner.com/programs.html`

## Bing Webmaster Tools
1. Open Bing Webmaster Tools property for `pub.jeremiahvanwagner.com`.
2. Go to `Sitemaps`.
3. Submit `https://pub.jeremiahvanwagner.com/sitemap.xml`.
4. Confirm sitemap crawl is successful.
